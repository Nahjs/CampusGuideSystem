项目名称：《交大校园导游咨询系统》
开发工具：Clion + QT Designer
主要功能：实现图的构造、图的遍历、求最短路径、求最小生成树以及查询景点信息
详细说明：
(1)有界面，显示了各个景点以及景点之间的路径和路径长度。
(2)构造图时选取了12个景点，图采用邻接矩阵存储。
(3)在文本框中输入起点编号，点击DFS按钮即可显示把所有景点都访问一遍的所有路径，不同的路径可以点击视图进行切换。
(4)在界面上输入起点后可在复选框选择终点，点击按钮可查询最短路径。
(5)实现用PRIM算法求最小生成树，在界面上用黄色铺设路径。
(6)鼠标点击某一景点时显示该景点的一些介绍信息和图片。
